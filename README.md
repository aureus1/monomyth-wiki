# Monomyth Wiki

The wiki-style database for **Monomyth** — the first-person fantasy RPG by
[Rat Tower](https://store.steampowered.com/app/908360/Monomyth/), built on
Unreal Engine 4.27 (`ProjectAriath`).

This repository is the public home for the wiki: it hosts the issue tracker for
reporting site bugs, data/location inaccuracies, and feature requests, plus the
documentation you're reading now.

## 🌐 Live site

The wiki is deployed at:

**👉 https://monomyth.aureus.me**

Browse weapons, spells, runes, armor, items, loot tables, status effects, and
interactive level maps with pinned item locations.

## 🗺️ What's on the wiki

- **Game Systems** — curated overviews of Weapons, Spells (including the
  Arx-Fatalis-style rune-trace casting system), Runes, and Armor.
- **Data Tables** — full browsable tables for ammo, consumables, crafting,
  loot tables, codes & keys, character stats, and status effects.
- **Level Maps** — in-game maps for each level with pinned world-item
  locations, projected from extracted game data.
- **Casting Guide** — how the pentagram rune-tracing spell system works.

## 🐛 Reporting issues

Found something wrong, or have an idea? Please open an issue — the templates
will guide you. Choose the one that fits:

| Type | When to use |
| --- | --- |
| [🌐 Location inaccuracy](.github/ISSUE_TEMPLATE/location_inaccuracy.md) | A pinned item/point is in the wrong spot on a level map |
| [🐛 Site bug](.github/ISSUE_TEMPLATE/bug_report.md) | Something on the website is broken (layout, links, search, rendering) |
| [✨ Feature request](.github/ISSUE_TEMPLATE/feature_request.md) | You'd like a new page, feature, or improvement |

> **A note on unreleased content:** Monomyth is still in development and the
> wiki may include unreleased items/levels. Unreleased content is hidden by
> default and can be toggled via the site-wide preference. If you spot
> something that looks like a spoiler leak, please flag it.
